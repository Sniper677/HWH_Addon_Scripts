# HeroWarsHelper Addon Scripts

This is the official documentation for the addon scripts designed to extend the functionality of the [HeroWarsHelper](https://greasyfork.org/en/scripts/450693-herowarshelper) tool.

## üöÄ Project Overview

This repository contains a collection of **Userscripts** (in the `.user.js` format) executed via browser extensions like **Tampermonkey** or **Violentmonkey**. These scripts provide additional features and quality-of-life improvements for the Hero Wars game.

## üõ† Prerequisites & General Installation

To use these scripts, you need:

1.  A compatible web browser (e.g., Chrome, Firefox, Edge).
2.  A Userscript Manager extension (e.g., [Tampermonkey](https://www.tampermonkey.net/), [Violentmonkey](https://violentmonkey.github.io/) or [ScriptRunner](https://chromewebstore.google.com/detail/ljajhcdgkhcdfcpkcmppdjanbdegggbp?utm_source=item-share-cb)).
3.  The [HeroWarsHelper](https://greasyfork.org/en/scripts/450693-herowarshelper) script from Greasy Fork website.

### ‚öôÔ∏è Installation Guide

1.  Install the Userscript Manager extension in your browser.
2.  Click on the link for the desired script in the [Script Index](#-script-index) below.
3.  The extension will prompt you to confirm the installation. Accept the installation.
